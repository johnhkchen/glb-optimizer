## Efficient Asset Distillation for Scene-Ready Rendering

### Problem

The plant model and gardening design pipeline (words -> research -> image -> model -> design tool) produces assets via Microsoft TRELLIS that look good but exceed triangle budgets by 10-40x for web/mobile scene rendering. Current decimation-only optimization leaves significant room for improvement. We need to distill high-detail models into representations that fit a low resource budget while preserving visual quality and — for hard-surface objects — providing construction-useful data.

### Scenario / Time Savings

**Current state:** Each asset requires manual optimization guesswork. TRELLIS models are decimated via ratio-based simplification, which degrades quality unpredictably and still often exceeds budgets. Billboard generation helps but is only one technique. No parametric reconstruction exists for known-geometry objects.

**Target state:** Assets entering the optimizer are automatically distilled using the most efficient representation strategy for their type:
- **Hard-surface / constructed objects** (planters, fences, trellises): Reconstructed from parametric primitives (dimensional lumber, standard shapes) with baked textures. Dual output: render-ready LODs + cut list / assembly data.
- **Organic / volumetric objects** (plants, flowers, foliage): Distilled via volumetric slicing — layered alpha-textured quads that reconstruct visual impression at minimal triangle cost. Tunable slice count for LOD.

**Time savings per asset:** Eliminates manual optimization iteration (currently 10-30 min per asset of tweaking ratios and inspecting results). Produces scene-ready output in one pass. Unblocks scene complexity — dozens of assets in a mobile scene instead of 3-5.

### Reference Assets

- `assets/wood_raised_bed.glb` (1.2M) — wooden raised bed, constructed from dimensional lumber
- `assets/rose_julia_child.glb` (7.8M) — Julia Child rose bush, dense organic foliage

### Tracks

**Track A — Hard Surface Reconstruction** (T-001-01, T-001-02)
Parametric primitive fitting for objects with known construction. Demonstrate with the raised bed: decompose into dimensional lumber boards, output efficient geometry + cut list.

**Track B — Organic Asset Distillation** (T-001-03, T-001-04)
Volumetric slicing / projection-based impostor generation for plants and foliage. Demonstrate with the Julia Child rose: generate layered textured quads that reconstruct the visual within extreme triangle budgets.

**Convergence — Scene Budget System** (T-001-05)
Per-asset-type budget allocation, pipeline integration, and output format for the design tool.
