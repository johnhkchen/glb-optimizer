## Context

T-004-01's spike validated (or pivoted from) the PCA approach. This ticket implements the production classifier: takes a GLB, extracts geometry features, returns a shape category and confidence score. The classification result is stored on the asset's settings so downstream code can read it.

## Acceptance Criteria

- New module: `scripts/classify_shape.py` (or Go integration via subprocess) per the spike's recommendation:
  - Input: path to a GLB file
  - Output: JSON with `{category, confidence, features}`
  - Categories: `round-bush`, `directional`, `tall-narrow`, `planar`, `hard-surface`, `unknown`
  - Features include: principal axes, eigenvalues, aspect ratio, density distribution
- New backend endpoint `POST /api/classify/:id` runs the classifier on the asset and stores the result
- Classification result added to `AssetSettings`: new fields `shape_category` (enum) and `shape_confidence` (float 0-1)
- Auto-classify on upload: when a new file is uploaded via `/api/upload`, run the classifier and persist the result
- Reclassify endpoint for existing files
- Classification emits a `classification` analytics event with `{category, confidence, features}` payload — this is critical training data
- The strategy router (T-004-03) reads the category from settings, so this ticket needs to land first
- Manual verification: classify the rose (expect `round-bush`), the raised bed (expect `hard-surface`), and a tall asset like a tree trunk (expect `tall-narrow`); confirm category + confidence appear in the asset's settings file

## Out of Scope

- Strategy router that uses the category (T-004-03)
- Multi-strategy comparison for ambiguous classifications (T-004-04)
- Branching topology (Y-shaped plants)
- Per-vertex semantic labels

## First-Pass Scope

The classifier doesn't need to be right 100% of the time. It needs to be right *enough* that the strategy router defaults are useful, and the multi-strategy comparison UI from T-004-04 catches the mistakes. Confidence threshold for triggering the comparison UI is conservative (~0.7) so unsure classifications get human review.
