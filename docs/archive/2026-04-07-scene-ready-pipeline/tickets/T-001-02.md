## Context

Once parametric geometry exists for the raised bed (T-001-01), the primitive boxes need convincing wood textures. The TRELLIS model and/or source images contain appearance information (wood grain, color variation, weathering) that should be transferred onto the simple geometry.

## Acceptance Criteria

- Bake or project textures from the original TRELLIS model onto the parametric reconstruction
- Support applying tileable wood textures with per-board UV variation (so boards don't look identical)
- Output uses a single shared texture atlas for the whole assembly (minimize draw calls)
- Visual comparison: side-by-side of original vs reconstructed model in the optimizer's preview
- Texture resolution appropriate for use case (512x512 or 1024x1024 atlas)
