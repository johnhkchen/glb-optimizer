## Context

The `assets/wood_raised_bed.glb` model (1.2M) is a TRELLIS-generated wooden raised bed. In reality, this object is constructed from dimensional lumber — standard boards (e.g., 2x6, 2x8, 4x4 posts) cut to length and fastened together. The TRELLIS mesh uses thousands of triangles to approximate what should be ~10-20 rectangular prisms.

The goal is to analyze the raised bed model, identify its constituent boards (dimensions, positions, orientations), and reconstruct it as a set of parametric primitives (boxes) with textures baked from the original model or source material.

## Acceptance Criteria

- Analyze `wood_raised_bed.glb` and identify the board components (count, approximate dimensions, arrangement)
- Generate replacement geometry from rectangular prisms matching standard dimensional lumber sizes
- Bake or apply wood textures to the primitive geometry
- Output a scene-ready GLB that is <50KB (vs 1.2M original) and visually recognizable as the same raised bed
- Output a cut list: board dimensions, quantities, lumber type (e.g., "4x 2x6x48in, 4x 4x4x24in")
- Demonstrate triangle count reduction (target: <300 triangles for the full assembly)
